h1. REX Blogmode

h3. Bloggen in der REDAXO-Struktur

Sinn & Zweck dieses Addons ist es, User einfach und unkompliziert neue Artikel anlegen zu lassen, ohne auf das OOFramework von Redaxo verzichten zu müssen. Wer immer dasselbe Modul auf Seiten in einer Kategorie legt, für den stellt dieses Addon in der täglichen redaktionellen Arbeit eine enorme Erleichterung dar.
Das heißt: Neue Artikel können ganz einfach mit einem Artikel angelegt, mit dem Content befüllt werden und online gestellt. 

h3. Einstellungen

Hier wird das Template und das Modul definiert, das im Blogmode bei neuen Artikeln zum Einsatz kommt. Zusätzlich müssen (!) hier die Kategorien ausgewählt werden, die beim Anlegen eines Artikels angeboten werden sollen.

h3. Mitgelieferte Module

Die beiden Module sind Beispiele, wie sinnvoll mit diesem Konzept umgegangen werden kann. Der Startartikel der gewählten Kategorie sollte sinnvollerweise das Modul "Blogmode Kategorieübersicht" enthalten. 

h3. Was das Addon nicht ist..

Das Addon ist zwar explizit dahingehend programmiert, daß es so übersichtlich und nachvollziehbar wie möglich gehalten ist, aber es ersetzt weder ein Tutorial noch eine ungefähre Vorstellung von Aufbau & Funktion eines Redaxo Addons..  und zumindest gewisse PHP/HTML Kenntnisse sind selbstredend. Sind diese Voraussetzungen erfüllt, dürfte man per copy, paste & adapt - _und einer eigenen Idee was man eigentlich machen will_ - aber schnell etwas funktionierendes auf solider Basis erstellen können.
